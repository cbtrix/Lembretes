# 🔔 Meus Lembretes

App de lembretes com notificações, filtros e prioridades — feito com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades

- Criar lembretes com título, data/hora, prioridade e observação
- Prioridades: Baixa, Média e Alta com cores distintas
- Filtros: Todos, Pendentes, Concluídos, Alta prioridade e Atrasados
- Marcar lembrete como concluído ou reabrir
- Destaque automático em vermelho para lembretes atrasados
- Notificações nativas do navegador no horário do lembrete
- Contadores no topo: total, pendentes e concluídos
- Dados salvos no `localStorage` (persistem ao fechar o navegador)
- Atalho: pressionar `Enter` no campo de título para adicionar

## 🛠️ Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — variáveis CSS, animações, layout responsivo
- **JavaScript** — DOM, localStorage, Notification API, setInterval

## 📁 Estrutura

```
lembretes/
├── index.html   # Estrutura da página
├── style.css    # Estilos e tema
└── script.js    # Lógica do app
```

## 🚀 Como usar

1. Clone ou baixe os arquivos
2. Abra o `index.html` no navegador
3. Clique em **Ativar** na barra de notificações para receber alertas
4. Adicione seus lembretes e filtre como quiser

> Não requer servidor. Funciona 100% offline.

## 📱 Responsivo

Funciona em desktop e mobile. Em telas pequenas o formulário adapta para coluna única.

## 🎨 Design

Tema claro com paleta quente em tons de terracota e creme. Tipografia: **Fraunces** (display) + **DM Sans** (corpo).
